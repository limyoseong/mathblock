# Issue #4: 문제 목록 + 진도 관리

**Type**: AFK
**Label**: ready-for-agent

## Parent

PRD.md - MathBlock 최소 수직 슬라이스

## What to build

5문제를 목록으로 보여주고, 문제를 선택해서 풀고, 진도를 추적하는 네비게이션 기능.

화면 흐름:
1. **문제 목록 화면** (메인): 5문제가 카드 또는 리스트로 표시. 각 문제마다 제목(KaTeX 수식), 풀었는지 여부(완료/미완료) 표시
2. 문제 클릭 → **풀이 화면**으로 이동 (Issue #1~#3에서 만든 화면)
3. 풀이 완료(정답) 후 → "다음 문제" 버튼으로 다음 문제로 이동 가능
4. 오답 시 → "다시 도전" 버튼으로 같은 문제 재시도 가능
5. "목록으로" 버튼으로 언제든 문제 목록 화면으로 돌아갈 수 있음

진도 저장:
- localStorage에 각 문제의 완료 상태 저장
- 브라우저를 닫았다 다시 열어도 진도가 유지됨

## Acceptance criteria

- [ ] 문제 목록 화면에서 5문제가 보인다
- [ ] 각 문제에 제목(수식)과 완료/미완료 상태가 표시된다
- [ ] 문제를 클릭하면 해당 문제의 풀이 화면으로 이동한다
- [ ] 정답 후 "다음 문제" 버튼이 나타나고 클릭하면 다음 문제로 이동한다
- [ ] 오답 후 "다시 도전" 버튼이 나타나고 클릭하면 워크스페이스가 초기화된다
- [ ] "목록으로" 버튼으로 문제 목록 화면으로 돌아갈 수 있다
- [ ] 문제를 풀고 목록으로 돌아오면 해당 문제가 "완료"로 표시된다
- [ ] 브라우저를 닫았다 다시 열어도 완료 상태가 유지된다
