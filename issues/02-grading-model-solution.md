# Issue #2: 채점 + 모범 풀이 표시

**Type**: AFK
**Label**: ready-for-agent

## Parent

PRD.md - MathBlock 최소 수직 슬라이스

## What to build

"실행" 버튼을 누르면 사용자의 블록 배치를 채점하고, 결과(정답/오답)와 모범 풀이를 보여주는 기능.

핵심 구현:
- `workspaceToTree()` 어댑터: Blockly의 `Blockly.serialization.workspaces.save()` 출력을 내부 solutionTree 포맷으로 변환. 이 어댑터가 Blockly와 앱 로직을 분리하는 핵심 인터페이스
- 채점 로직: 사용자 트리의 최종 답(루트 노드의 expectedOutput)과 문제의 expectedAnswer를 비교하여 정답/오답 판정
- 결과 화면: 정답이면 "정답!" 표시. 오답이면 "오답" 표시 + 모범 풀이 블록 배치를 사용자 풀이 옆에 나란히(side-by-side) 표시
- 각 블록 단계의 중간 결과값(expectedOutput)을 블록 옆에 표시

에러 처리:
- 빈 워크스페이스에서 실행 → "블록을 배치해 주세요" 안내 메시지
- 연결되지 않은 블록 → 무시하되 "연결되지 않은 블록이 있습니다" 경고

평가 모델은 룩업 기반. 런타임 수식 계산 없음. 모든 중간값과 최종 답은 문제 데이터에 사전 정의되어 있다.

## Acceptance criteria

- [ ] "실행" 버튼이 화면에 있고 클릭 가능하다
- [ ] 블록을 올바르게 배치하고 실행하면 "정답!" 메시지가 표시된다
- [ ] 블록을 잘못 배치하고 실행하면 "오답" 메시지가 표시된다
- [ ] 오답 시 모범 풀이 블록 배치가 사용자 풀이 옆에 나란히 보인다
- [ ] 각 블록 단계의 중간 결과값이 표시된다
- [ ] 빈 워크스페이스에서 실행하면 안내 메시지가 나온다
- [ ] 연결되지 않은 블록이 있으면 경고가 표시된다
- [ ] `workspaceToTree()` 어댑터에 대한 테스트가 존재하고 통과한다
- [ ] 채점 로직에 대한 테스트가 존재하고 통과한다
